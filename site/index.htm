AppTools

AppTools is a next-generation package management and application deployment suite. It contains tools and utilities for building, installing and removing AppFS packages.

AppTools is unique in that it solves problems at every level of the application deployment stack and seamlessly integrates all package management components of the system, whether they are the utilities used in AppTools, or existing package management systems such as RPM or Debian.

To find out more, check out our Why? and What? pages. Alternatively, to download the source and get more developer-orientated documentation, check out the Google Code repository.
